# Money Manager (Mini Project)

Minimal full-stack app:
- Backend: Node.js + Express + SQLite
- Frontend: Vanilla HTML/CSS/JS (minimalist)

## Run backend
1) cd backend
2) npm install
3) copy .env.example to .env (optional)
4) npm run dev

Backend runs at:
- http://localhost:5050/api/health
- http://localhost:5050/api/transactions
- http://localhost:5050/api/summary

Frontend is served automatically at:
- http://localhost:5050/
